脚本安装要求：
1. 安装 Python 3.6
2. 安装 opencv-python
3. 安装 pyautogui
4. 安装 pydirectinput

实现方法：
1. 基于pyautogui, pydirectinput实现坐标定位，鼠标点击和键盘输入
2. 基于opencv实现图形识别

系统要求：
1. Windows 10 / 11
2. 中文系统

备注：
1. Windows 11中图标不同，需要手动替换部分识别的图标，保证脚本顺利运行
2. 无盘和有盘环境都可以运行
